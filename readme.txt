Connect AutomatorWP with GamiPress

== Description ==

> **Important:** Now all free integrations are included in [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") so you no longer need to install them one by one!
>
> We will continue working to improve AutomatorWP and make it easier, faster and more accessible to everyone.

[GamiPress](https://wordpress.org/plugins/gamipress/ "GamiPress") is the most powerful gamification plugin that let's you award your users with digital rewards for interacting with your WordPress website.

This plugin automatically connects [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") with GamiPress adding new triggers that you can use to connect with other plugins and automate workflows letting you save time and get focused on your most important work.

= Triggers =

* User earns any/specific amount of points of any/specific type.
* User earns any/specific achievement.
* User reaches any/specific rank.

= Actions =

* Award points to user.
* Award achievement to user.
* Award rank to user.

= Pro version =

[Pro version](https://automatorwp.com/add-ons/gamipress/ "AutomatorWP - GamiPress") available that let's you expand the integration between AutomatorWP and GamiPress bringing you more ways to automatize your website gamification.

= What is AutomatorWP? =

AutomatorWP is an automator plugin that lets you connect your WordPress plugins together and add automation to unlimited workflows.

You can automate anything including sales, marketing, administrative tasks, learning and any other kind of processes you want letting you save time and get focused on your most important work.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
